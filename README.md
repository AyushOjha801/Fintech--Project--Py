# 📈 Equal Weight Indian Stock Portfolio Strategy

This project builds a simple equal-weight stock portfolio using the top Indian companies by market capitalization.

Using Python and Yahoo Finance, the program fetches live stock data, ranks stocks based on market cap, selects the top 10 companies, and calculates how many shares to buy for each stock based on the investment amount entered by the user.

## 🔧 Technologies Used
- Python
- Pandas
- NumPy
- yFinance
- Jupyter Notebook

## 🚀 Features
- Fetches live stock prices and market caps
- Selects the top 10 Indian stocks
- Implements an Equal Weight Strategy
- Calculates shares to purchase for each stock
- Easy to customize and extend

## 📊 Example
If you invest ₹10,000, the program divides the amount equally among the selected stocks and determines the number of shares that can be purchased for each.
